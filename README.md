# final-review — Claude Code 考前突击 Skill

期末复习 / 考前突击专用 skill，自动按剩余天数切换三种教学模式，零基础也能跟上。

## 安装

```bash
# 用户级（所有项目生效，推荐）
mkdir -p ~/.claude/skills/final-review/
cp SKILL.md ~/.claude/skills/final-review/

# 项目级（仅当前项目生效）
mkdir -p .claude/skills/final-review/
cp SKILL.md .claude/skills/final-review/
```

重启 Claude Code 即可生效。

## 触发方式

直接说出以下任意关键词：

- **紧急**：`期末复习`、`考前突击`、`明天考`、`临时抱佛脚`
- **常规**：`复习助手`、`刷题`、`下周考`、`备考`
- **英文**：`midterm`、`final`
- **进阶**：`知识图谱`、`速记卡`

## 三种模式

| 剩余天数 | 模式 | 策略 |
|----------|------|------|
| ≤3 天 | 🚨 紧急 | 真题驱动 + 只讲高频 + 速记卡 |
| 4-7 天 | 📋 标准 | 真题讲解 + 重点练习 + 模拟考 |
| >7 天 | 📚 系统 | AI 出题 + 全量刷题 + 错题本 |

## 教学特色

**三栏教学法：**

```
真题长这样 → 考场上写这个 → 换成XX还是同一套
```

每道题并排对比，标注 🔒 锁死的骨架（直接抄）和 🔓 跟着题目变的部分。

**讲解风格：**
- 先直觉后定义（生活类比 → 正式定义）
- 不讲底层原理，只讲怎么写
- 每个知识点配"失分提醒"
- 讲完立刻出同类题验证

## 用途边界

这是一个**考前短期突击工具**，不是长期学习助手。

- ✅ 期末/期中/考研冲刺（考前 1-14 天）
- ✅ 零基础紧急备考（"明天考今天才开始"）
- ❌ 日常系统学习
- ❌ 学术论文写作
- ❌ 代码开发任务
